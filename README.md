# Sistema Full Stack de Gestión de Empleados

## Descripción

Proyecto Full Stack desarrollado utilizando Django, Django REST Framework, HTML, CSS y JavaScript. El sistema permite gestionar empleados mediante operaciones CRUD conectadas a una API REST.

---

# Tecnologías utilizadas

* Python
* Django
* Django REST Framework
* HTML
* CSS
* JavaScript
* SQLite
* GitHub

---

# Funcionalidades

* Registro de empleados
* Visualización dinámica
* Edición de empleados
* Eliminación de empleados
* Buscador dinámico
* Validaciones de formularios
* Interfaz responsive

---

# Arquitectura del sistema

Frontend (HTML/CSS/JS)
↓
API REST Django
↓
Base de datos SQLite

---

# Desarrollo por fases

## Fase inicial

Configuración del entorno de desarrollo e instalación de dependencias.

## Fase 1

Implementación de la consulta y visualización de empleados.

## Fase 2

Desarrollo del formulario de registro de empleados.

## Fase 3

Implementación de actualización de empleados.

## Fase 4

Desarrollo de eliminación de registros.
